How you get the data

Get data from Factory, Service, Provider - Allow us to encapsulate common functionality.

The difference between the three is just the way in which they create the object that goes and gets the data

  • Factory: Create an object of the factory and return it
  • Service: Standard function that uses the this keyword to define
  • Provider: $get you define and it can be used to get the ojbect that return the data
